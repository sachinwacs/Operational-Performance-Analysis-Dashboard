# Operational Performance Analysis Dashboard

## Project Overview
This Operational Performance Analysis Dashboard provides a comprehensive overview of representatives’ performance for the call centre from January 2023 to December 2023.

## Objectives
- Evaluate call count, call duration, Sales Income(Amount), ratings for each representative.
- Rank representatives via their call count, Sales Income(Amount)

## Dataset
The dashboard uses the following datasets:  
- call data
- customer data

## Tools & Technologies
- **Power Query** – For data cleaning, transformation, and preparation  
- **Power Pivot** – For data modeling
- **DAX**(such as CALCULATE, SUM)– For calculating measures
- **Excel formulas**(such as IF, XLOOKUP)
- **Excel Charts** – For Visualizing Data

## Key Metrics & KPIs
- Call Count - 1000
- Call Duration – 89,850 Minutes
- Amount - $96,623
- Average rating - 3.9
- Call Rank
- Amount Rank

## Dashboard Features
- KPI summary displaying Call Count, Call Duration, Amount, Average rating, Call Rank, Amount Rank
- Line chart displaying trends of call count by month
- Bar chart displaying trends of call count by day of week
- Column chart visualising male vs female customer calls by city
- Bar chart displaying call count by representative and amount by representative
- Column chart visualising customer rating for representatives
- Table showing representative’s sales according to city & customer
- All visuals can be filtered by representatives
